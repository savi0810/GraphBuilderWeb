# GraphBuilderWeb

Я написал графический калькулятор, который переводит выражение от х в график функции на холсте.

## Структура решений
* GraphBuilderWeb - ASP.NET Web Api (апи парсит выражение и отдаёт точки на клиент)
* GraphBuilderFrontend - Blazor Web Assembly (клиентская часть приложения: меню, холст, панель управления)
* GraphBuilderShared - обычная библиотека классов c# (создал для хранения общих моделей т.к. почему-то не получилось создать ссылку на проект между апи и клиентом)


## Логика и технологии
  
  ### Api

  для парсинга я использовал библиотеку mXparser. Говоришь ей аргумент и она сама считает от него, функций не так много ,но все основные она парсит нормально

  ### Client

  тут основная технология это SVG (Scalable Vector Graphics). У компонента svg есть атрибут path который принимает путь в виде " M 100 200 L 150 250 L 200 300..." который нужноо отрисовать. M (Move to) - откуда начать рисовать, т.е. начальная координата. L (Line to) соответственно следующая координата с которой должна соединиться предыдущая. Основная задача была в том чтобы подогнать точки, который отдаёт апи под координаты холста, эта задача конечно не трудна в реализации, но это моё первое веб-приложение и я нормально так помучился + постарался сделать интересный функционал, ну добавить ещё очень много чего было в планах.


## Функционал

  * при открытии страницы сразу отрисовывается тестовая функция параболы
  * можно добавлять новую функцию на холст
  * можно удалить любую функцию из списка
  * есть выбор цвета функции при добавлении либо рандомно либо можно выбрать самому
  * есть возможность менять цвет функции, которая уже добавлена
  * есть возможность менять саму формулу добавленной функции, график меняется динамически
  * есть панель с примерами функций, нажимаешь и она появляется на графике
  * есть панелька управления видом холста, кнопки приближения и отдаления а также кнопка, которая сбрасывает вид до дефолтных настроек
  * есть возможность регулировать зум не только кнопками, но и колесом мыши
